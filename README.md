# MediatRDemo.Api
This repo focuses on providing .NET 6 microservice template with boilerplate setup. 
* MediatR to implement Middleware for **Validation**, **Logging** (Telemetry), and **Exception** Handling


## Patterns
* CQRS
* MediatR

## Tech Stack
* .NET 6
* MediatR
* Fluent Validations
